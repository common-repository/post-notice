Wordpress Post Notice

== Description ==

开发这个插件的主要原因是因为目前我在管理一个团队，但是发文章的时候会需要一些特别的注意事项，口头上虽然和编辑们说了，但是仍然有人不记得，所以希望编辑们在写文章的时候，可以有一个公告时刻叮嘱他们。所以就开发了这个插件。对于一些SEO主管，应该或多或少有些帮助。。
